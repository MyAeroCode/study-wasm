
### WASM - C Examples

---

##### How to test it?

1. open web-server on this dir.

```shell
python3 -m http.server
```

<br/>

2. append url param `idx` with index number.

```shell
0.0.0.0:8000/?idx=01
```

<br/>

3. It's all. Open the debug console.
